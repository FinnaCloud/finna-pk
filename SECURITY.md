# Security Policy

## Reporting a Vulnerability

**IMPORTANT: Do not file a public issue or ticket** mentioning the vulnerability or security issue.

### Bug Bounty Program Exclusion

**IMPORTANT: This project is explicitly EXCLUDED from our bug bounty reward program.**

We do not offer bug bounties, rewards, or compensation for security reports related to finna-pk.

### How to Report

To privately report security vulnerabilities or issues, please send your report to:

**noc@finnacloud.com**

**Please note: This project is excluded from our bug bounty reward program. No rewards or compensation will be provided.**

### What to Include

Your security report should include:

- **Summary**: A clear description of the security issue
- **Steps to Reproduce**: Detailed steps needed to reproduce the vulnerability
- **Impact Assessment**: The potential security impact of the issue
- **Mitigation**: If you have identified any proposed mitigation or fix, please include it

### Response Time

We will acknowledge receipt of your report within 48 hours and provide an initial assessment within 7 business days.

### Bug Bounty Policy

**This project is explicitly EXCLUDED from our bug bounty reward program.**

We appreciate responsible disclosure of security vulnerabilities, but please be aware that:

- **No bug bounties are offered** for this project
- **No rewards or compensation** will be provided for security reports
- **This exclusion applies** to all security vulnerabilities reported for finna-pk

### Additional Guidelines

In addition to the above, the [Reporting Guidelines](https://finnacloud.com/vulnerability-disclosure) from our vulnerability disclosure policy apply to all security reports.

**Reminder: This project is excluded from our bug bounty reward program.**

### Responsible Disclosure

We appreciate responsible disclosure of security vulnerabilities. We ask that you:

- Allow us a reasonable amount of time to address the issue before public disclosure
- Not access or modify data that does not belong to you
- Not perform any actions that could harm our users or systems

**Please note: No bug bounties, rewards, or compensation are available for this project.**

Thank you for helping keep finna-pk secure.
